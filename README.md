# QR Portal Card

Neon, glassy take on the QR component. The QR and CTA both open my GitHub profile.

![Screenshot of the QR portal card](./images/screenshot.png)

## Live
- Open locally: `index.html`
- Target link: https://github.com/OluochBen
- Favicon: `./images/favicon-32x32.png`

## Build
- Semantic HTML + modern CSS (flexbox, gradients, glassmorphism)
- Google Fonts: Space Grotesk, Inter
- Mobile-first layout with narrow-width tweaks

## How to use
1) Clone/download this repo.  
2) Open `index.html` in your browser.  
3) Scan the QR or click the button to reach the profile.

## Notes
- The design diverges from the original challenge; restricted design previews stay out of git via `.gitignore`.
- Accessibility: linked QR has descriptive `alt` text and label.
